Leetcode Question 76: Minimum Window Substring

Explanation Solution

Minimum Window Substring

Given two strings, original and check, return the minimum substring of original such that each character in check, including duplicates, are included in this substring. By "minimum", I mean the shortest substring. If two substrings that satisfy the condition have the same length, the one that comes lexicographically first is smaller.

Examples

Example 1

Input: original = "CDBAEBAECD", check = "ABC"

Output: BAEC

Explanation: BAEC is the shortest substring of original that contains all of A, B, and C.

Example 2

Input: original = "ADOBECODEBANC", check = "ABC"

Output: BANC

Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.

The solution is similar to Find Longest Subarray with Sum Smaller Than Target; it's a find longest sliding window problem.

In this case, the comparison for checking a valid window is changed to compare that for every character in check, to see if the window contains enough of that character.

We could apply the find longest sliding window template to this question. To check the validity of a window, we introduce two variables (counters): required and satisfied.

required: the number of distinct characters we are required to have. Counter for distinct characters that appeared in check.

satisfied: the number of distinct characters that are satisfied within the current window. Now, we can check whether required == satisfied to confirm if the window is valid.

Since we only care about those characters that appear in check, we will only make changes to satisfied when the added/removed character appears in check. If the current window contains exactly the same amount of one specific character as in check after adding the counter, then we increase satisfied by 1 (we will not update satisfied on all other additions). If we remove a character from the window and that character is part of check, we check whether the amount has dropped below check's amount.

Time Complexity: O(n)

Space Complexity: O(n)
