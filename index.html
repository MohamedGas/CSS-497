Minimum Window Substring

Given two strings, original and check, return the minimum substring of original such that each character in check, including duplicates, are included in this substring. By "minimum", I mean the shortest substring. If two substrings that satisfy the condition have the same length, the one that comes lexicographically first is smaller.

Examples

Example 1

Input: original = "CDBAEBAECD", check = "ABC"

Output: BAEC

Explanation: BAEC is the shortest substring of original that contains all of A, B, and C.

Example 2

Input: original = "ADOBECODEBANC", check = "ABC"

Output: BANC

Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.

Explanation

Intuition

Use HashMap to take a count of every element of t.

Then use Two pointer on s to match result.

Approach

i is used to take character and j is used to release characters.

Lets add all occurrences of character in t.

for(int i = 0; i < t.length(); i++){
    char ch = check.charAt(i);
    hm2.put(ch, hm2.getOrDefault(ch, 0) + 1);
}

Now let's take two pointers i and j and count variable mcnt.

So I will increase my count if my HashMap of t (count of particular variable) is greater than HashMap of s (count of that particular variable).

char ch = original.charAt(i);
if(hm1.getOrDefault(ch, 0) <= hm2.getOrDefault(ch, 0)){
    mcnt++;
}

After this part, now it is time to calculate the minimum one.

So, I will make a String potential answer (pans) and the substring will be from j + 1 to i + 1.

If pans.length is smaller than answer.length, then it means it has the minimum length, so I will store it in ans. And I will increase j to find more smaller.

Now it is time to release character, so we will check if frequency is 1 then directly remove from HashMap else decrease the frequency by one.

if(hm1.get(ch) == 1) hm1.remove(ch);
else hm1.put(ch, hm1.get(ch) - 1);

Now check if HashMap for s is lesser than HashMap for t, then decrease count. It means we can still get a lesser length of the string.
